Flat Search
============

## Usage

```bash
Flat Search
Usage:
  fs --version
  fs --help
  fs <olx_search_url> <file_path> [--format <format>] [--from <from>] [--to <to>]

Options:
  --help, -h                      Show this screen
  --version, -v                   Show the current app version
  --format <format>, -f           Output file format [default: txt]
  --from <from>                   Start page [default: 1]
  --to <to>                       Finish page
```
